    class Person

    Instances of this class represent people with data like name or address.

    Attributes

    address[RW]

    The person's address (an Address instance).

    email[RW]

    The person's email address (a String).

    name[RW]

    The person's name (a Name instance).

    phone[RW]

    The person's phone number as a String.

    new( name, address, phone, email ) click to toggle source

    Constructor. The name parameter could be a Name or a String instance. If it is a String the Name constructor Name.new will be called providing this String as the parameter. This delegates the split into given name and surname to the Name class.

    # File person.rb, line 33
def initialize( name, address, phone, email )
  @name = name.kind_of?( Name ) ? name : Name.new( name )
  @address = address
  @phone = phone
  @email = email
end

    to_hash() click to toggle source

    Return the data for this person as a Hash with sub Hash elements. For example the value for the key 'name' is a Hash with the keys title, givenname, and surname.

    # File person.rb, line 65
def to_hash()
  {
    :name => {
      :title => self.name.title,
      :givenname => self.name.givenname,
      :surname => self.name.surname },
    :address => {
      :housenumber => self.address.house_number,
      :street => self.address.street,
      :postalcode => self.address.postal_code,
      :town => self.address.town
    },
    :phone => self.phone.to_s,
    :email => self.email
  }
end
    to_html( person_css_class = '', name_css_class = '' ) click to toggle source

    Return the data for this Person in HTML, formatted as one paragraph (P) with several lines (using the BR tag for separation). The optional first argument is the name of a CSS class for the whole paragraph. The optional second argument is the name of a CSS class to be used in a SPAN element around the full name of the person.

    # File person.rb, line 52
def to_html( person_css_class = '', name_css_class = '' )
  "<p class=\"#{person_css_class}\">\n" +
  "<span class=\"#{name_css_class}\">#{@name.to_s}</span><br />\n" +
  # "#{@address.street}<br />#{@address.town}<br />\n" +
  @address.to_html + "<br />\n" +
  "#{@phone.to_s}<br />\n" +
  %Q(<a href="mailto:#{@email}" title="Email to #{name.to_s}" class="email">#{@email}</a>\n) +
  "</p>\n"
end

    Return the complete data set for this person as a String. Should look like what you expect in a letter head or so.

    # File person.rb, line 42
def to_s()
  @name.to_s + "\n" + @address.to_s + "\n" + @phone.to_s + "\n" + @email.to_s
end

    Return the data for this person in XML format. The XML uses no attributes and the structure corresponds to the Hash structure returned by #to_hash.

def to_xml()
  XmlSimple.xml_out( { :person => self.to_hash() }, { :KeepRoot=>true, :NoAttr=>true } )
end
